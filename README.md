# Web Video Clipper Pro

Aplikasi web untuk memotong video secara online tanpa perlu menginstal apapun.

## Fitur

- ✅ Upload video (MP4, MOV, AVI, MKV)
- ✅ Preview video
- ✅ Pemotongan manual & otomatis (scene, face, warna)
- ✅ Preset filter: slow motion, speed up, grayscale, reverse
- ✅ Export ke MP4, GIF, MP3
- ✅ Login sederhana

## Teknologi

- [Streamlit](https://streamlit.io)
- [MoviePy](https://zulko.github.io/moviepy/)
- [OpenCV](https://opencv.org/)
- [SceneDetect](https://pyscenedetect.com/)

## Cara Menjalankan Lokal

```bash
pip install -r requirements.txt
streamlit run web_video_clipper_advanced.py
